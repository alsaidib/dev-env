# rollup-starter-dev-env

[![Greenkeeper badge](https://badges.greenkeeper.io/dalisoft/rollup-starter-dev-env.svg)](https://greenkeeper.io/)

Starter Dev-Env with React, Babel+Buble and Rollup

# Running

First, don't forget:
```bash
npm install
```

### Development

```bash
npm start
```

### Production

```bash
npm run build
```

# Test

```bash
npm test
```
